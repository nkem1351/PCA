DESCRIPTION
This program performs Principal Component Analysis (PCA) for identifying
essential variables for securing donor funding at Anderson Cancer Center.

HOW TO RUN
-Install Python
1. Install Python 3.8 or higher.
2. Install an IDE like Visual Studio Code or PyCharm.
5. Run 'data_analyst.py'.

-Install R.
1. Install R.
2. Install an IDE like RStudio.


DEPENDENCIES 
Python: NumPy, Pandas, and Matplotlib libraries are required.
R: No external libraries are required.
